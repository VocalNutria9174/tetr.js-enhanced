### Default Controls:

Controls are configurable

- **Rotate Left:** Z
- **Rotate Right:** Up Arrow
- **Rotate 180:** Shift
- **Hold:** M
- **Hard Drop:** Space
- **Shift Left:** Left Arrow
- **Shift Right:** Right Arrow
- **Restart:** R
